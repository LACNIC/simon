Proyecto Simón

Mapa de interconexión entre redes de América Latina y Caribe a través de mediciones de latencia.

El mapa de interconexión entre redes de LAC se elabora gracias a la colaboración de la comunidad de entusiastas de Internet, y se nutre de mediciones de latencia realizadas desde los medidores java (ntp), traceroute (icmp), y javascript (http).

Objetivos

El proyecto Simón tiene el objetivo de determinar la interconexión entre diferentes redes de América Latina y el Caribe a través de mediciones de latencia entre distintos puntos.
