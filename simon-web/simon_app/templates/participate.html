¿Cómo Participar?

Existen varias maneras de participar de Simon! La forma más fácil de participar es realizar pruebas de latencia desde tu país. Además, se puede participar inscribiendo tu servidor Web o NTP a la lista de Servidores de Prueba. Para registrar tu servidor, contáctanos a través de proyectosimon@lacnic.net

Alojando el medidor javascript en tu sitio

Basta agregar la siguientes líneas de código en alguna página de tu sitio para que los visitantes comiencen a disparar mediciones de latencia en la región LAC. Este mecanismo permite generar mediciones de forma "mesh", es decir desde muchos orígenes hacia muchos destinos. 

/*
 * Include the libraries (jQuery needed)
 */

/*
 * Include the script in an asyn way.
 */
<script type="text/javascript">
(function (d, s) {
        var js = d.createElement(s),
                sc = d.getElementsByTagName(s)[0];

        js.src = "http://simon.lacnic.net/simon/static/simon_app/js/simon_probe_plugin.js";
        js.type = "text/javascript";
        sc.parentNode.insertBefore(js, sc);

        js.onload = js.onreadystatechange = function () {
                // Whenever you wish to start the tests, call SIMON.init()
            SIMON.init();
        };

    }(document, "script"));
</script>
{# #}

Este tester depende de las librerías jQuery. Si no se encuentran disponibles en la página donde resida el script de Simón, puedes incluirlas a través de una línea.

<script src="http://code.jquery.com/jquery-1.11.0.min.js"></script>

Algunas notas:

  • El medidor es transparente al usuario y no interfiere con su navegación normal en el sitio.
  • Es posible ajustar la cantidad de usuarios que generan mediciones. Por ejemplo, se puede configurar para que el 10% del tráfico en tu sitio genere mediciones de Simón. Esta función es conveniente si quieres ir aumentando gradualmente tu colaboración con el proyecto.

Colaborando con tu servidor de pruebas

Es de suma importancia disponer de la mayor cantidad de servidores posibles. Cuanto más servidores estén disponibles para realizar los test, podemos tener mayor cantidad de información y datos más certeros.
Si conoces o tienes un servidor NTP, o dispones de un servidor web a lso cuales se les puede hacer un HTTP GET. La única condición que deben cumplir es que se encuentre en la región de Latino América o el Caribe.
