¿Cómo Participar?

Existen varias maneras de participar de Simon, en particular, la forma mas fácil es realizar los test de latencia desde tu país.

Medidores de Latencia y Ancho de Banda

Ejecutando uno de los medidores y esperando que finalicen los test, nos estas ayudando a recolectar importante información de la conectividad de tu país, que nos servirá para conocer el estado de la interconexión en la región.
Existen dos tipos de medidores:

  • Java Applet para medir latencia NTP a múltiples puntos (esperar a que las pruebas finalicen)
  • Javascript para medir latencia TCP a múltiples puntos de prueba
  • Javascript latency lab para medir latencia TCP a una dirección IP en particular

Servidores de Test

Es de suma importancia disponer de la mayor cantidad de servidores posibles. Cuanto más servidores están disponibles para realizar los test, podemos tener mayor cantidad de información y datos más certeros.
Si conoces o tienes un servidor NTP, o dispones de un servidor web donde puedas hostear algunos archivos, nos ayudarías muchísimo. La única condición que deben cumplir es que se encuentre en la región de Latino América o el Caribe.

Servidor de mediciones de latencia

Basta agregar las siguientes líneas de código en alguna página de tu sitio para que los visitantes comiencen a disparar mediciones de latencia en la región LAC. este mecanismo permite generar mediciones de forma "mesh", es decir desde muchos orígenes hacia muchos destinos. Para ver más información acerca del medidor Javascript lee estas notas.

Algunas notas:

  • El medidor es transparente al usuairo y no interfiere con su navegación normal en el sitio
  • Es posible ajustar la cantidad de usuarios que generan mediciones. Por ejemplo, se puede configurar para que el 10% del tráfico en tu sitio genere mediciones de Simón. Esta función es conveniente si quieres ir aumentando gradualmente tu colaboración con el proyecto
