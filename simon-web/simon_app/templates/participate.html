¿Cómo Participar?

Existen varias maneras de participar de Simon! La forma más fácil de participar es realizar pruebas de latencia desde tu país. Además, se puede participar inscribiendo tu servidor Web o NTP a la lista de Servidores de Prueba. Para registrar tu servidor, contáctanos a través de proyectosimon@lacnic.net

Medidores de Latencia

Existen dos tipos de medidores:

  • Java Applet para medir latencia NTP a múltiples puntos (esperar a que las pruebas finalicen)
  • Javascript para medir latencia TCP a múltiples puntos de prueba
  • Javascript latency lab para medir latencia TCP a una dirección IP en particular

Servidores de Prueba

Es de suma importancia disponer de la mayor cantidad de servidores posibles. Cuanto más servidores están disponibles para realizar los test, podemos tener mayor cantidad de información y datos más certeros.
Si conoces o tienes un servidor NTP, o dispones de un servidor web a lso cuales se les puede hacer un HTTP GET. La única condición que deben cumplir es que se encuentre en la región de Latino América o el Caribe.
